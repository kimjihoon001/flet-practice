# Flet 기반 공학용·사무용 전환형 계산기 구현

## 🚀 Introduction
Flet을 활용해 구현한 멀티모드 계산기로,
사무용 계산기와 공학용 계산기를 전환하며 사용할 수 있도록 설계했다.

## 🔑 Features

### 🧮 Standard Mode
- 사칙연산 (+, −, ×, ÷) 및 퍼센트 기능
- 괄호 연산 지원

### 🛠️ Engineering Mode
- 삼각함수: sin, cos, tan (DEG/RAD 지원)
- 로그함수: log, ln
- 지수함수: eˣ, xʸ
- 상수: π, e
- 부호전환 지원
- 제곱, 루트, 절댓값 등 지원
  
### Additional Features
- 공학용/사무용 전환 기능
- 자동 괄호 닫힘 처리
- 백스페이스 기능
